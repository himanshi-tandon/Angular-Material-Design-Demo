Matterial Buttons we have seven type of buttons

1) at-button Rectangular text button w/ no elevation
2) mat-raised-button Rectangular contained button w/ elevation
3) mat-flat-button Rectangular contained button w/ no elevation
4) mat-stroked-button Rectangular outlined button w/ no elevation
5) mat-icon-button Circular button with a transparent background, meant to contain an icon
6) mat-fab Circular button w/ elevation, defaults to theme's accent color
7) mat-mini-fab Same as mat-fab but smaller
